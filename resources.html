Navio Core

Navio Core is the foundational open-source software that powers the Navio blockchain. It is the core node implementation responsible for consensus, block validation, transaction processing, and network-level operations. Navio Core is based on a fork of Bitcoin Core, adapted for Navio’s privacy-focused protocol and Proof-of-Private-Stake (PoPS) consensus mechanism.

Navio Typescript SDK

TypeScript SDK for interacting with the Navio blockchain. Provides wallet management, transaction key synchronization, BLSCT confidential transaction support, and blockchain interaction through Electrum servers or P2P connections.
